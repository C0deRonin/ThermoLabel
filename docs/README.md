## 💡 Основные концепции

### v0.3.0 включает:

✅ **Полное тестирование** - Jest с 73+ тестами  
✅ **PostgreSQL интеграция** - SQLAlchemy с миграциями Alembic  
✅ **Сохранение проектов** - localStorage с меню ProjectsMenu  
✅ **Загрузка без палитры** - ImageLoadModal с двумя режимами  
✅ **Светлая тема** - System CSS переменных  
✅ **Русский интерфейс** - i18n с 80+ переводами  
✅ **Исправления багов** - State persistence, canvas issues
