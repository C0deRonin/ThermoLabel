# Roadmap

## Текущий фокус

- Повышение стабильности интерфейса разметки.
- Расширение backend API для управления проектами.
- Улучшение UX при работе с крупными изображениями.

## Ближайшие задачи

1. Улучшение производительности рендера аннотаций.
2. Расширение форматов экспорта.
3. Улучшение мониторинга ошибок и логирования.
4. Усиление покрытия тестами критичных сценариев.

## Среднесрочные задачи

- Ролевая модель доступа.
- Импорт/экспорт конфигураций проекта.
- Расширенная аналитика по датасету.

## Долгосрочные задачи

- Интеграции с внешними ML-пайплайнами.
- Поддержка пакетной обработки изображений.
