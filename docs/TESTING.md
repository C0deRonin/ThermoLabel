# Тестирование

## Цели

Тестирование в Infralabel подтверждает корректность:
- frontend-сервисов;
- backend API;
- бизнес-логики работы с проектами и настройками.

## Frontend

Используются:
- Jest;
- Testing Library.

Команды:
```bash
npm test -- --runInBand
npm run lint
```

Основной набор тестов расположен в `__tests__/`.

## Backend

Используются:
- Pytest;
- тестовые фикстуры из `backend/tests/conftest.py`.

Пример запуска:
```bash
cd backend
pytest
```

## Практика

- Пишите unit-тесты для сервисов.
- Добавляйте проверки граничных случаев.
- Для исправлений багов добавляйте тест, воспроизводящий проблему.
- Проверяйте линтеры перед публикацией изменений.
