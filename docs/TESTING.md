# 🧪 Тестирование ThermoLabel

Приложение и тесты запускаются **только через Docker**. См. README.md.

## ✅ Быстрая проверка (Quick Start Test)

### 1. Запуск приложения

```bash
docker compose up -d
# Откройте http://localhost:3000 в браузере
```

### 2. Базовые проверки

#### ✓ Интерфейс загружается
- [ ] Видна левая панель с инструментами
- [ ] Видна центральная область с холстом
- [ ] Видна правая панель со списком аннотаций
- [ ] Видна верхняя панель с контролами

#### ✓ Загрузка изображения (демо)
- [ ] На холсте видно тепловое изображение
- [ ] Изображение разноцветное (палитра Iron)
- [ ] Размер отображается как 640×480
- [ ] Все палитры работают (нажимайте на палитры в левой панели)

#### ✓ Инструменты аннотации

**Bounding Box:**
- [ ] Выберите инструмент "▭ Bounding Box"
- [ ] Выберите класс (Перегрев)
- [ ] Нарисуйте прямоугольник на изображении
- [ ] Аннотация появилась в правой панели
- [ ] На холсте видна цветная рамка

**Полигон:**
- [ ] Выберите инструмент "⬡ Полигон"
- [ ] Кликайте на изображение 5+ раз
- [ ] Видите счетчик точек в левой панели
- [ ] Нажмите "Закрыть" или двойной клик на первую точку
- [ ] Полигон создан и отображается на холсте

**Пороговый инструмент:**
- [ ] Выберите инструмент "⚡ Порог"
- [ ] Передвигайте слайдер
- [ ] На холсте белыми пиксели выве порога
- [ ] Нажмите "Применить авто"
- [ ] В правой панели появляются новые bbox'ы

#### ✓ Информация о курсоре
- [ ] Наведите мышь на изображение
- [ ] В левой панели появляются X, Y, T значения
- [ ] Значения обновляются при движении

#### ✓ Гистограмма
- [ ] Видна гистограмма в левой панели
- [ ] Это похоже на цветной график

## 💾 Тестирование экспорта

### YOLO Export

```bash
1. Создайте несколько аннотаций
2. Нажмите "↓ YOLO"
3. Должен скачаться файл annotations.txt
```

**Проверка содержимого:**
```
0 0.5 0.5 0.3 0.4
1 0.7 0.3 0.2 0.2
```

- [ ] Каждая строка это одна аннотация
- [ ] Формат: `<class_id> <x_norm> <y_norm> <w_norm> <h_norm>`
- [ ] Значения 0-1 (нормализованные)

### COCO Export

```bash
1. Нажмите "↓ COCO"
2. Должен скачаться файл annotations_coco.json
```

**Проверка содержимого (JSON):**
- [ ] Содержит массив "images"
- [ ] Содержит массив "annotations"
- [ ] Каждая аннотация имеет "thermal" объект с mean/max/min
- [ ] Валидный JSON (откройте в любом JSON viewer)

### Pascal VOC Export

```bash
1. Нажмите "↓ Pascal VOC"
2. Должен скачаться файл annotations.xml
```

**Проверка содержимого (XML):**
```xml
<annotation>
  <filename>thermal_demo.jpg</filename>
  <width>640</width>
  <height>480</height>
  <objects>
    <object>
      <name>Перегрев</name>
      <bndbox>
        <xmin>100</xmin>
        <ymin>100</ymin>
        <xmax>200</xmax>
        <ymax>200</ymax>
      </bndbox>
      <thermal>
        <mean>65.50</mean>
        <max>70.00</max>
        <min>60.00</min>
      </thermal>
    </object>
  </objects>
</annotation>
```

- [ ] Валидный XML
- [ ] Содержит правильные bbox координаты
- [ ] Температурные данные присутствуют

## 🔄 Тестирование истории

### Undo функция

```bash
1. Создайте несколько аннотаций
2. Нажмите кнопку ↩ (Undo)
```

- [ ] Последняя аннотация удалится

### Очистить всё

```bash
1. Создайте несколько аннотаций
2. Нажмите "✕ Очистить всё" в правой панели
```

- [ ] Все аннотации удалены
- [ ] Это действие можно отменить через Undo

## 🎨 Тестирование классов

### Добавить класс

```bash
1. Нажмите вкладку "Классы"
2. Введите "Новый класс" в поле
3. Нажмите на цветной квадрат и выберите цвет
4. Нажмите "+ Добавить"
```

- [ ] Новый класс появился в списке
- [ ] Новый класс доступен в левой панели для выбора
- [ ] При создании аннотаций используется этот класс

### Удалить класс

```bash
1. На вкладке "Классы" нажмите ✕ рядом с классом
```

- [ ] Класс удален из списка
- [ ] Класс больше недоступен для выбора

## ⚙️ Тестирование Backend API (опционально)

### Запуск backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### Проверка здоровья

```bash
curl http://localhost:8000/api/health
```

**Ожидаемый ответ:**
```json
{"status": "ok", "version": "0.2.0"}
```

- [ ] Ответ 200 OK
- [ ] Версия правильная

### Форматы

```bash
curl http://localhost:8000/api/supported-formats
```

- [ ] Список форматов возвращается
- [ ] JSON корректный

### Обнаружение аномалий

```bash
curl -X POST http://localhost:8000/api/detect-anomalies \
  -H "Content-Type: application/json" \
  -d '{
    "annotations": [
      {"tempStats": {"mean": 25.0}},
      {"tempStats": {"mean": 26.0}},
      {"tempStats": {"mean": 80.0}}
    ]
  }'
```

- [ ] Ответ 200 OK
- [ ] Обнаруживается аномалия с средней температурой 80
- [ ] JSON правильный
