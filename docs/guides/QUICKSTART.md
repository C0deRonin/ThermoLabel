# Quickstart

## Требования

- Node.js и npm.
- Python 3.11+.
- PostgreSQL (для локального backend) или Docker.

## Быстрый запуск frontend

```bash
npm install
npm run dev
```

## Запуск полного стека через Docker

```bash
docker-compose up --build
```

## Проверка качества

```bash
npm run lint
npm test -- --runInBand
```

## Следующий шаг

После запуска перейдите к `docs/guides/USING_GUIDE.md`.
