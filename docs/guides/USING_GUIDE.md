# Руководство по использованию

## 1. Запуск

1. Запустите приложение локально или через Docker.
2. Откройте интерфейс в браузере.

## 2. Базовый сценарий разметки

1. Загрузите термальное изображение.
2. Создайте или выберите класс объекта.
3. Нарисуйте аннотацию нужным инструментом.
4. При необходимости отредактируйте границы.
5. Сохраните результат.

## 3. Управление проектом

- Создавайте отдельные проекты для разных наборов данных.
- Используйте именование классов с понятной предметной областью.
- Регулярно проверяйте статистику по аннотациям.

## 4. Экспорт

- Выберите формат экспорта.
- Убедитесь, что у всех нужных объектов есть корректные классы.
- Экспортируйте датасет для дальнейшего использования в ML-пайплайне.

## 5. Рекомендации

- Используйте undo/redo для безопасного редактирования.
- Проверяйте контраст палитры для сложных изображений.
- Делайте резервные копии проектов при длительной работе.
