# Скриншоты для документации

Файлы в этой папке используются в корневом `README.md` и в `docs/README.md`.

## Имена файлов (v1.0.0)

| Файл | Содержание |
|------|------------|
| `01-annotation.jpg` | Главный экран: аннотация, bbox, палитра, классы, список аннотаций |
| `02-projects-modal.jpg` | Меню «Сохранённые проекты» |
| `03-db-dump.jpg` | Окно «Дамп базы данных» (экспорт/импорт SQL) |
| `04-analytics.jpg` | Вкладка «Аналитика»: графики, распределение классов |
| `05-classes.jpg` | Вкладка «Классы»: управление классами |
| `06-annotation-detail.jpg` | Дополнительный вид рабочей области (при необходимости) |

## Как заменить на свои скриншоты

1. Сделайте снимки экрана интерфейса (PNG или JPEG; для README лучше PNG).
2. Переименуйте файлы по таблице выше и **положите в эту папку**, заменив существующие.
3. Закоммитьте изменения — на GitHub скриншоты отобразятся в README автоматически.

> В репозитории могут лежать компактные заглушки; замените их полноразмерными скриншотами (PNG/JPEG) для читаемой документации.
